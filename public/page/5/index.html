Getting domain record details (NS, MX, TXT, SOA, SPF, A, AAAA, any) By default, dig only provides you wih an A record, i.e. an IP. To get details about a specific record, you need to pass it as an option with dig, like so: dig NS aamnah.com dig aamnah.com NS dig MX aamnah.com dig NS aamnah.com is the same as dig aamnah.com NS. It doesn’t matter if you pass the option in the beginnnig or at the end.
