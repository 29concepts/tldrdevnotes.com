Delete ALL Unapproved comments in WordPress

01 Jan 0001, 00:00
wordpress / mysql

Make a backup first before changing anything. Go to PhpMyAdmin and select the comments table (usually wp_comments). Go to the SQL tab and run the following command. DELETE FROM wp_comments WHERE comment_approved = 0 That’s all.
